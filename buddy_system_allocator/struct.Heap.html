Expand description

A heap that uses buddy system with configurable order.

Usage

Create a heap and add a memory region to it:

use buddy_system_allocator::*;
let mut heap = Heap::<32>::empty();
unsafe {
    heap.init(begin, size);
    // or
    heap.add_to_heap(begin, end);
}
